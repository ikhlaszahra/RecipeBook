# 🍲 Recipe Book Web Application

A simple full-stack web application for managing recipes. Users can add, view, edit, and delete recipes easily with a clean and responsive interface.

---

## 🚀 Features

* Add new recipes with ingredients and instructions
* View all saved recipes
* Edit existing recipes
* Delete unwanted recipes
* Simple and user-friendly interface
* Database-backed storage (MySQL)

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/recipe-book.git
```

2. Move project to `htdocs` (if using XAMPP)

3. Start Apache & MySQL in XAMPP

4. Import database:

* Open phpMyAdmin
* Create a database (e.g. `recipe_db`)
* Import provided `.sql` file

5. Run project in browser:

```
http://localhost/recipe-book/
```
